# Qubit Roll 🎲🌌

**Qubit Roll** is a premium, quantum-themed puzzle-action game where you control a state-shifting cube (a "Qubit") through a series of increasingly complex grid-based levels. Master the laws of quantum physics to navigate obstacles and eliminate enemies.

---

## 🚀 Experience the Quantum Realm

In **Qubit Roll**, your movement is your power. As you roll the cube, its faces rotate, changing your available abilities. Strategy lies in positioning the right "quantum state" on top to trigger powerful effects.

### ✨ Core Quantum Mechanics

| Ability | Face Color | Description |
| :--- | :--- | :--- |
| **Heisenberg's Dash** | 🔴Red | Instantly slide 2 tiles. Essential for striking down Red Enemies. |
| **Schrödinger's Split** | 🔵Blue | Divide into two entangled cubes. Be in two places at once to tackle complex puzzles. |
| **Quantum Tunneling** | 🟢Green | Phase directly through solid walls into the adjacent space. |

### 👾 The Adversaries
- **Red Enemies**: Standard drones. Can only be killed via a **Heisenberg Dash**.
- **Cyan Enemies**: Entangled entities. They can **only** be eliminated while you are in a **Split** state.

---

## 🎮 How to Play

### Controls
- **WASD / Arrow Keys**: Roll the cube across the grid.
- **Q / E**: Rotate the cube in place to align your faces.
- **SPACE**: Activate the ability of the current **Top Face**.
- **R**: Restart the level if you reach a "No-Win" state or die.
- **ESC**: Return to the level selection menu.

### Objective
Navigate each level, utilize your quantum abilities strategically, and eliminate all enemies to unlock the next challenge.

## 🛠️ Technical Overview

Qubit Roll is built with modern web technologies, prioritizing smooth performance and rich visual aesthetics.

- **Engine**: Pure Vanilla JavaScript (ES6+).
- **Rendering**: HTML5 Canvas API for high-performance 2D graphics.
- **Styling**: Vanilla CSS3 with glassmorphism effects and dynamic CSS transitions.
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts for a sleek, modern UI.

### Key Features
- **Dynamic Face Mapping**: Realistic 3D-to-2D face rotation logic.
- **Entanglement System**: Global state synchronization for split cubes.
- **Particle System**: Custom visual feedback for dashes, tunneling, and kills.
- **Responsive Layout**: Designed to look premium on various screen sizes.

## 📥 Getting Started

To play the game locally:
1. Clone the repository or download the source files.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
3. No build step or local server required!
